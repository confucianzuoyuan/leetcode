\section{题目}

给你一个链表, 删除链表的倒数第 `n` 个结点, 并且返回链表的头结点. 

![](images/19-1.jpg)

\textbf{示例 1}: 

```
输入: head = [1,2,3,4,5], n = 2
输出: [1,2,3,5]
```

\textbf{示例 2}: 

```
输入: head = [1], n = 1
输出: []
```

\textbf{示例 3}: 

```
输入: head = [1,2], n = 1
输出: [1]
```

**提示**: 

- 链表中结点的数目为 `sz`
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

**进阶**: 你能尝试使用一趟扫描实现吗？
