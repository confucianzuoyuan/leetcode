\section{题目}

给你一个按照非递减顺序排列的整数数组 `nums`, 和一个目标值 `target`. 请你找出给定目标值在数组中的开始位置和结束位置. 

如果数组中不存在目标值 `target`, 返回 `[-1, -1]`. 

你必须设计并实现时间复杂度为 $O(logN)$ 的算法解决此问题. 

\textbf{示例 1}: 

```
输入: nums = [5,7,7,8,8,10], target = 8
输出: [3,4]
```

\textbf{示例 2}: 

```
输入: nums = [5,7,7,8,8,10], target = 6
输出: [-1,-1]
```

\textbf{示例 3}: 

```
输入: nums = [], target = 0
输出: [-1,-1]
```

**提示**: 

- $0 <= nums.length <= 10^5$
- $-10^9 <= nums[i] <= 10^9$
- `nums` 是一个非递减数组
- $-10^9 <= target <= 10^9$
